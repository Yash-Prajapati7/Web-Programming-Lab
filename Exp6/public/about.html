Understanding the Node.js HTTP Module

What is This?

This is a demonstration of a basic web server built using Node.js's built-in http module. It shows how HTTP requests and responses work at a fundamental level, without relying on frameworks like Express.

How It Works

  1. Request Handling: When you request a URL, the server receives an HTTP request object.
  2. File Resolution: The server maps the URL to a file path in the public/ directory.
  3. File Reading: Node.js's fs module reads the file from disk.
  4. Response: The server sends back the file content with appropriate headers.
